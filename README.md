# BitAngleMondulation-with-Interrupts
Bit Angle Modulation - per Interrupt gesteuert

Ein kleines Beispielprogramm mit dem man eine BAM "Bit Angle Modulation" mit Interrupts/Countern erzeugen kann.
An Port A können die LEDs an PIN 0+1 angeschlossen werden. Sie werden parallel angesteuert, es würde also auch eine LED ausreichen um die Funktion zu beobachten.
Das Programm läuft auf einem ATTiny26 mit 8 Mhz und wurde mit AVR-Studio 6.2 kompiliert.
